## Installation

```sh
$ make
```

Install tmux plugins, this is currently manual.

```sh
$ tmux
# prefix-key, I
```

Restart your shell

exec -l $SHELL

Run shellcheck tests

```sh
$ make test
```

## Help

Sometimes in tmux you need to update the GPG_TTY

```sh
$ export GPG_TTY=$(tty)
```
